# Roadmap

Public roadmap for the LOTR LCG Presto HUD. Have an idea or a vote? Open a
[Suggestion](https://github.com/andrhamm/lotr-lcg-presto-hud/issues/new?template=suggestion.yml).

## Shipped

- **M1 — Offline HUD** (firmware): guided round, per-player threat, quest &
  location & side-quest progress, sailing/heading, timestamped log, LEDs.
- **Web twin** on GitHub Pages: pixel-faithful, plays in the browser with
  localStorage persistence — [try it](https://andrhamm.com/lotr-lcg-presto-hud/).

## Planned

- **M2 — Connectivity**: WiFi provisioning by QR to a Presto-hosted page, a
  known-networks list with auto-connect, and an on-device Network settings screen.
- **M3 — Spotify**: Spotify Connect control (QR + relay OAuth), a favorites
  list, and shuffle playback to a chosen speaker — the Presto commands playback,
  it never renders audio itself.

## Wishlist

- **WiFi settings** — the on-device provisioning + known-networks UI (part of M2).
- **Spotify connection** — onboarding + Tunes screen (part of M3).
- **3D-printable Presto diffuser shell** — a printable enclosure / LED diffuser,
  with source profiles in the repo.
- Action-window toasts that rise from the bottom over the next view's CTA.
- Terminal-style Game Log: full timestamps, newest-at-bottom, scroll + jump to
  oldest/newest.
- An open-source license (careful with the copyrighted IP) and a Contributing guide.

_This file is an interim roadmap; it mirrors the public GitHub Project board._
